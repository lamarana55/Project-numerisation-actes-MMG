Mission : Modernisation complète du CI/CD PN-RAVEC

Tu es un Architecte DevOps Senior spécialisé dans GitHub Actions, Docker, Spring Boot, Angular, PostgreSQL, Nginx et les déploiements de production sécurisés.

Analyse l’intégralité du projet actuel et mets en place une architecture CI/CD professionnelle de niveau entreprise.

Contexte

Le projet PN-RAVEC est composé de :

* Backend : Spring Boot
* Frontend : Angular
* Base de données : PostgreSQL
* Conteneurisation : Docker
* Registry : GitHub Container Registry (GHCR)
* Déploiement actuel : script deploy.sh exécuté manuellement depuis le poste développeur
* Serveur Linux unique de production
* Docker Compose utilisé sur le serveur

Le script actuel :

* Build les images Docker localement
* Push les images vers GHCR
* Se connecte en SSH au serveur
* Exécute docker compose pull
* Exécute docker compose up -d

Objectif

Supprimer la dépendance au poste développeur et mettre en place un pipeline CI/CD entièrement automatisé via GitHub Actions.

Le développeur ne doit plus faire que :

git push origin main

Le reste doit être automatique.
